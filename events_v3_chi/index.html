What is this?
A live map of upcoming events in 64 Chicago neighborhoods — north side, downtown, and the south side (where source coverage is still building out) — pulled automatically from venue calendars, library systems, parks feeds, and ticketing platforms. Each event is scored by an LLM on things a person actually cares about — can I show up alone? will I talk to strangers? does going regularly build something? — and recurring events (a weekly knitting circle, a monthly civic meeting) are flagged as ↻ joinable, because community comes from showing up to the same thing twice. Click a neighborhood to focus it; click a pin for its calendar. The ranked list has the same data as a sortable table, and how this works explains the engine behind it.
